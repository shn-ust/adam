# ADAM ![](https://github.com/shn-ust/adam/actions/workflows/go.yml/badge.svg)

Find dependencies of an application in a distributed environment.

## Getting started

### Building the image

`$ docker build -t adam .`

### Running the image
`$ docker run --network host -it adam`
